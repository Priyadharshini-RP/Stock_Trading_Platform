# Stock_Trading_Platform

A console-based **Stock Trading Platform** developed in **Java** that simulates a basic stock market environment. The application allows users to register, log in, view market data, buy and sell stocks, manage their portfolio, and track transaction history using Object-Oriented Programming (OOP) concepts.

## Features

* User Registration and Login
* View Available Stocks and Market Prices
* Buy and Sell Stocks
* Portfolio Management
* Track Portfolio Performance (Stock Value, Cash Balance, Net Worth)
* Transaction History
* Input Validation for Invalid Operations
* Menu-Driven Console Interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* HashMap
* Scanner (Console Input)

## OOP Concepts Implemented

* Classes and Objects
* Encapsulation
* Constructors
* Collections Framework (`ArrayList`, `HashMap`)
* Modular Programming using Multiple Classes

## Project Structure

* `Stock` – Stores stock name and current price.
* `Transaction` – Records buy and sell transactions.
* `User` – Manages user details, portfolio, balance, and transaction history.
* `TradingSystem` – Handles registration, login, market operations, portfolio management, and transactions.
* `TradingApp` – Main class to run the application.

## How to Run

1. Compile the program:

   ```
   javac TradingApp.java
   ```
2. Run the application:

   ```
   java TradingApp
   ```
